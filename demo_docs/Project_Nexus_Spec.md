# Project Nexus：高空太陽能無人機 (HALE) 技術規格書
**版本：** v2.1 (2026 內部機密)
**產品代號：** NX-2000 "SkyWatcher"

## 1. 系統概述
Project Nexus 旨在建立一個能夠在同溫層持續飛行超過 90 天的自動化無人機網路。該系統利用高轉換效率的柔性鈣鈦礦太陽能薄膜（Perovskite Solar Cells）作為主要動力來源。

## 2. 核心技術參數
*   **翼展 (Wingspan):** 32.5 公尺
*   **最大起飛重量 (MTOW):** 165 公斤
*   **巡航高度 (Cruise Altitude):** 18,000 公尺至 22,000 公尺 (同溫層)
*   **推進系統:** 4 具高效率無刷直流馬達 (BLDC)，每具輸出功率為 1.5 kW。
*   **能源儲存系統:** 
    - 主要電池：三元鋰電池組 (NMC)，能量密度達 450 Wh/kg。
    - 總儲備容量：85 kWh。

## 3. 通訊與負載
*   **負載重量:** 可攜帶最大 15 公斤的通訊天線組。
*   **下載速率:** 支援 6G 太赫茲 (THz) 頻段，骨幹網傳輸速度達 200 Gbps。
*   **控制延遲:** 衛星對地通訊延遲需低於 12 毫秒 (ms)。

## 4. 異常處理與安全協議 (重要測試點)
*   **協定 A-12 (電力短缺):** 當電池電量低於 15% 時，系統必須自動關閉非必要的 6G 通訊模組，並將飛行姿態轉為「最大滑翔效率模式」。
*   **協定 B-09 (突發氣流):** 若偵測到超過 45 節的陣風，無人機應立即下降至 15,000 公尺的安全層位。
*   **維護週期:** 每運作 5,000 個小時需進行一次整機結構超音波檢測。

## 5. 法律與合規性
本系統符合《2025 全球平流層飛行器管理規範》第 4.2 條，具備自動防撞系統 (TCAS-X)。
